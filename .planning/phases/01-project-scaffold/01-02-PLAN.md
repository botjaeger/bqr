  <action>
    Create components/canvas/DrawingCanvas.tsx as a "use client" component. It must:

    1. Declare a ref: `const canvasRef = useRef<HTMLCanvasElement>(null)`.
    2. Track drawing state with `useRef<boolean>(false)` (isDrawing) — do NOT use useState for this; it avoids re-renders on every mouse event.
    3. Implement three mouse event handlers attached to the canvas element:
       - onMouseDown: set isDrawing.current = true, call ctx.beginPath(), ctx.moveTo(e.offsetX, e.offsetY)
       - onMouseMove: guard with `if (!isDrawing.current) return`; call ctx.lineTo(e.offsetX, e.offsetY), ctx.stroke()
       - onMouseUp / onMouseLeave: set isDrawing.current = false
    4. Set canvas drawing style in a useEffect on mount: ctx.strokeStyle = "#000000", ctx.lineWidth = 2, ctx.lineCap = "round", ctx.lineJoin = "round".
    5. Render a <canvas> element with: ref={canvasRef}, width={600}, height={600}, className="border border-border rounded-md bg-white cursor-crosshair", and the four event handlers above.
    6. Wrap in a <div> with className="flex flex-col items-center gap-4".
    7. Include a "Clear" button (import Button from "@/components/ui/button") that calls ctx.clearRect(0, 0, canvas.width, canvas.height) on click.

    Important pitfalls to avoid:
    - Mark file with "use client" at the top — canvas refs and mouse events are client-only.
    - Do NOT use devicePixelRatio scaling yet; that is introduced in Phase 3 alongside the QR renderer. Keep coordinates simple.
    - Do NOT use useState for isDrawing — the mouse move handler fires at 60fps and setState would cause 60 re-renders/second.
    - offsetX/offsetY are correct for mouse position relative to the canvas element when there is no CSS transform/scale on the canvas.

    Create app/canvas/page.tsx as a server component (no "use client" needed — it just imports and renders the client component):
    - Import DrawingCanvas from "@/components/canvas/DrawingCanvas"
    - Render: `<main className="flex min-h-screen flex-col items-center justify-center gap-4 p-8"><h1 className="text-2xl font-bold">Canvas</h1><DrawingCanvas /></main>`

    Update app/page.tsx: add a Next.js <Link href="/canvas"> wrapping the existing Button, so the user can navigate to the canvas. Import Link from "next/link". Change Button text to "Open Canvas".
  </action>
